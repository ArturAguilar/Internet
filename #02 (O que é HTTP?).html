O que é HTTP?

Definição

O HTTP (HyperText Transfer Protocol) é um protocolo usado para a comunicação entre navegadores e servidores na web. Ele permite a transferência de informações como páginas da web, imagens, vídeos e muito mais.

Como funciona?

O HTTP segue um modelo de requisição e resposta:

  • Requisição: O navegador solicita informações ao servidor, como ao acessar um site.
  • Resposta: O servidor envia os dados solicitados, como o conteúdo de uma página web.

Características

  • Baseado em texto: As requisições e respostas são feitas em texto simples.
  • Sem estado: Cada interação é independente; o servidor não mantém informações sobre requisições anteriores.
  • Porta padrão: O HTTP usa a porta 80 por padrão.

Exemplo de Requisição HTTP

GET /index.html HTTP/1.1
Host: www.exemplo.com

Essa é uma solicitação HTTP pedindo o arquivo index.html do servidor www.exemplo.com.

HTTP vs HTTPS

O HTTPS é a versão segura do HTTP, onde as informações trocadas são criptografadas. Ele usa o protocolo SSL/TLS para proteger os dados.
